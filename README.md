# Project Name
talkdoc_core

## Installation

1. **Clone the repository**
    ```bash
    git clone https://github.com/foundersai25/talkdoc_core
    cd talkdoc_core
    ```

2. **Install talkdoc_core **
    ```bash
    pip install .
    ```

3. **Run the Streamlit app**
    ```bash
    streamlit run Chat.py
    ```
