# Project Name

A Streamlit-based chat application.

## Installation

1. **Clone the repository**
    ```bash
    git clone https://github.com/your-username/your-repo-name.git
    cd your-repo-name
    ```

2. **Install talkdoc_core **
    ```bash
    pip install .
    ```

3. **Run the Streamlit app**
    ```bash
    streamlit run Chat.py
    ```
